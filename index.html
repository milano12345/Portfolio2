Datify Success App

I collaborated with a team on this app. Our data scientist used a crawler to pull data from the OKcupid app. Then we used TDIF vectorization to turn strings into numbers and graphed the result to view visible clusters of data. We then created an algorithm to compare user input to the data and return a number between 1-100 to tell you how similar your profile description is to others. I also help created the sign up form's on the Front-end.

Front-end: HTML5, Sass, React, React-Router, Netlify
Back-end: Node, Express, Next, SQLite3, Heroku
