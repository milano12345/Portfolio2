My Recent Work

Here are a few design projects I've worked on recently. Want to see more? Email me.

Anywhere Fitness App

Clients and instructors have the ability to register accounts. Passwords are encrypted and stored on the database. Instructor's have full CRUD ability, while clients have read-only access. Client's can also register for future classes once logged in.

Front-end: HTML5, CSS, Styled-Components, React, React-Router, Netlify
Back-end: Node, Express, Next, Bcrypt, JWT, Knex, SQLite3, Heroku Learn more

Datify Success App

I collaborated with a team on this app. Our data scientist used a crawler to pull data from the OKcupid app. Then we used TDIF vectorization to turn strings into numbers and graphed the result to view visible clusters of data. We then created an algorithm to compare user input to the data and return a number between 1-100 to tell you how similar your profile description is to others. I also help created the sign up form's on the Front-end.

Front-end: HTML5, Sass, React, React-Router, Netlify
Back-end: Node, Express, Next, SQLite3, Heroku Learn more

Stripe Catalogue

This is a custom store with a shopping cart similar to that of Shopify. The goal is to market the site to potential clients and be able to process the client orders.

Front-end: HTML5, CSS, EJS, Javascript, Netlify
Back-end: Node, Express, Next, Knex, SQLite3, Stripe, Heroku Learn more
